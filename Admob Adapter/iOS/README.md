# Kidoz Admob Adapters

To add Kidoz Admob Adapter to your project with CocoaPods:
```
pod 'Google-Mobile-Ads-SDK','10.8.0'
pod 'KidozSDK', '9.0.0'
pod 'KidozAdmobAdapter', '1.0.1'
```
